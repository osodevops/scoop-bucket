# OSO DevOps Scoop Bucket

A [Scoop](https://scoop.sh/) bucket for OSO tools.

## Usage

```powershell
# Add the bucket
scoop bucket add osodevops https://github.com/osodevops/scoop-bucket

# Install kafka-backup
scoop install kafka-backup
```

## Available Apps

| App | Description |
|-----|-------------|
| [kafka-backup](https://github.com/osodevops/kafka-backup) | High-performance Kafka backup and restore with point-in-time recovery |
